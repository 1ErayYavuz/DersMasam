</div>

---

## ✨ Özellikler

| Özellik | Açıklama |
|---------|----------|
| ⏱️ **Odak Sayacı** | 1-300 dakika arası özelleştirilebilir çalışma süresi |
| 💧 **Su Hatırlatma** | Her 30 dakikada bir su içme hatırlatması |
| 🔊 **Doğal Sesler** | Su damlalarını taklit eden 3 farklı melodi |
| 📝 **Görev Yönetimi** | Post-it tarzı görev ekleme ve silme |
| 🎁 **Ödül Sistemi** | Çalışma sonunda kendine ödül belirle |
| 💪 **Motivasyon** | Kişisel motivasyon cümlenle çalış |
| 💾 **Oturum Kaydetme** | Yarım kalan çalışmalar kaldığı yerden devam eder |
| 🎨 **Modern Tasarım** | Koyu tema, glassmorphism efektleri |
| 📱 **Mobil Uyumlu** | Telefon ve tabletlerde sorunsuz çalışır |
| 🔇 **Sessiz Mod** | Ses ve hatırlatmaları açıp kapatabilirsin |

## 📦 Nasıl Kullanılır

### 1. Projeyi İndir

```bash
git clone https://github.com/kullanici-adi/DersMasam.git
cd DersMasam
```

### 2. Aç

`index.html` dosyasını herhangi bir tarayıcıda açmanız yeterlidir. Kurulum gerektirmez!

> 💡 **İpucu:** Dosyayı çift tıklayarak doğrudan tarayıcıda açabilirsiniz.

### 3. Kullan

1. **Çalışma süresini** gir (1-300 dakika)
2. **Motivasyon cümleni** yaz (isteğe bağlı)
3. **Ödülünü** belirle (isteğe bağlı)
4. **"MASAYI KUR"** butonuna tıkla
5. Odaklanmaya başla! 🎯

## 🎵 Ses Sistemi

Proje, Web Audio API kullanarak ses dosyaları olmadan doğal sesler üretir:

### Bitiş Melodisi
Süre dolduğunda çalan 6 saniyelik yumuşak kutlama melodisi.

### Su Sesleri
Her 30 dakikada bir çalan 3 farklı su melodisi:

| Melodi | Frekans | Efekt |
|--------|---------|-------|
| 🫧 Damlama | 1100-1600 Hz | Yüksek frekanslı su damlaları |
| 🌊 Akarsu | 700-1200 Hz | Yükselip alçalan dalga |
| 💦 Su Perdesi | 660-1100 Hz | Harmonik çakışma |
